## Requirements

Install the dependencies with pip

```
Pillow==9.2.0
streamlit==1.12.0
```

## Run the app

Terminal

```
# vanilla terminal
streamlit run app.py
```

## Run automagically

Find `.devcontainer/_devcontainer.json` have a look inside, remove underscore and get through the enchanted gate to github codespaces automation.
